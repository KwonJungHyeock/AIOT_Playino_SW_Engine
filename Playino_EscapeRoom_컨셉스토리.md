# Playino : Escape Room — 컨셉 & 스토리라인
**명세서(게임콘텐츠_4챕터_상세)의 "스토리 레이어" — 각 방의 intro/clear를 이 서사로 교체. 퀘스트·미니게임·핀·learning은 그대로.**

## 0. 한 줄 컨셉
어둠 속 폐연구소에 홀로 깨어난 EDDIE. **방마다의 미션(커리큘럼)을 풀어 시스템을 복구하며 시설을 탈출**한다. — 학습 미션 = 탈출 퍼즐.

## 1. 톤 가이드 (중요 · K-12)
- **으스스한 미스터리** 수준: 폐연구소, 깜빡이는 비상등, 정적, 낮은 험, 먼지·그림자.
- **금지**: 고어·폭력·유혈·인물 위협·실제 공포 연출.
- 위협은 **추상적 장치**로만(봉인·타이머·어둠·침수). EDDIE는 용감·호기심, 클리어 시 안도·성취.
- 어둠 속 **EDDIE 눈 글로우가 유일한 빛** → 시그니처 강화. 클리어하면 그 구역만 환해진다.

## 2. 세계관 & 스토리 아크
- **오프닝**: EDDIE가 칠흑 같은 연구소에서 홀로 부팅된다. 문은 잠겼고 전원은 꺼졌다. "여기서… 나가야 해."
- **진행**: 복도(HUB)의 잠긴 방을 하나씩 풀어 **전력·보안코드·열쇠**를 모으고 시스템을 복구한다. 풀수록 복도가 조금씩 밝아진다.
- **클라이맥스**: 모든 시스템 복구 → 메인 보안문 해제.
- **엔딩**: 문이 열리고 빛이 쏟아진다. EDDIE 탈출 — 자유. (떡밥: 누가, 왜 가뒀나? → 시즌2)
- **타이틀 카드**: EDUINO AI 부팅 스플래시 → `Playino : Escape Room` 타이틀 → 오프닝.

## 3. 챕터 = 탈출 막(ACT)
| 챕터 | 탈출 막 | 의미 |
|---|---|---|
| CH1 | **각성** | EDDIE 부팅·시스템 연결 = 첫 생존 신호 |
| CH2 | **시스템 복구** | 기초 방을 풀어 전력·단서 확보 |
| CH3 | **보안 돌파** | 조합 장치로 더 강한 잠금 해제 |
| CH4 | **최종 탈출** | 메인 시스템 복구 → 출구 개방 |

## 4. 방별 탈출 리스킨
> learning(부품·개념·퀘스트·미니게임)은 명세서 그대로. **상황·클리어만 컨셉화.** 보상 = 탈출 진행 요소.

### CH1 · 각성
- **보드 준비** → "어둠 속 각성": 꺼진 보드를 깨워 EDDIE 기동. 연결 성공 = 첫 불빛/생존 신호 → 탈출 시작.

### CH2 · 시스템 복구 (LED·DHT11 고정)
- **LED** 🔒고정 — 프레이밍: "암흑 속 첫 불빛" — 꺼진 시설의 조명을 켜 시야 확보. *(learning·내용 그대로)*
- **DHT-11** 🔒고정 — 프레이밍: "생명 유지 진단" — 시설 환경(온·습도)을 측정해 안전 확인. *(그대로)*

| 방 | 탈출 상황(장치) | 클리어 보상 |
|---|---|---|
| RGB LED | **색 암호등** — 특정 색에서만 열리는 문 패널, 색을 맞춰 봉인 해제 | 색 코드 조각 |
| 부저 | **경보 해제** — 울리는 경보, 멜로디 시퀀스를 맞춰 정지 | 정적 회복 |
| 7세그먼트 | **봉인 카운트다운** — 방을 잠그는 숫자 카운트, 멈춰서 해제 | 출구 일부 개방 |
| 7세그 4칸 | **금고 코드** — 4자리 금고, 반응속도로 코드 입력 | 금고 속 열쇠 |
| 8×8 매트릭스 | **벽의 신호** — 깜빡이는 벽 패널 패턴 재현 | 숨은 통로 표식 점등 |
| 스텝 모터 | **기어 잠금** — 녹슨 밸브/기어를 정확한 각도로 회전 | 봉인문 개방 |
| 릴레이 | **차단기 복구** — 꺼진 구역의 차단기를 올림(딸깍) | 구역 전력 복구 |
| 가변저항 | **주파수 다이얼** — 잡음 콘솔의 다이얼을 맞춤 | 단서 수신 |
| 조도 센서 | **빛 봉인** — 빛에 반응하는 봉인, 어둠/빛 조절로 통과 | 봉인 통과 |
| 소리 감지 | **정적의 방** — 소리에 반응하는 함정, 정확한 순간에만 신호 | 함정 통과 |
| 수위 센서 | **침수 탈출** — 차오르는 물, 수위를 조절해 잠기기 전 탈출 | 배수 성공 |
| 불꽃 감지 | **화재 구역** — 통로를 막은 불씨를 감지해 안전 경로 확보 | 경로 확보 |
| 키패드 | **출입문 비번** — 잠긴 문의 비밀번호를 추리 | 출입문 해제 |

### CH3 · 보안 돌파
| 방 | 탈출 상황 | 보상 |
|---|---|---|
| 리모컨 컨트롤 | **원격 제어실** — 멀리 있는 장치를 리모컨으로 조작해 길을 연다 | 원격 통로 개방 |
| 디지털 시계 | **시간 봉인문** — 특정 시각에만 열리는 문, 시계를 맞춤 | 시간문 개방 |
| 자동 조명 | **어둠의 복도** — 자동 점등 시스템을 복구해 어둠 통과 | 복도 점등 |
| 화재 경보기 | **화재 차단** — 불 감지+경보로 막힌 출구를 정리 | 출구 정리 |
| 스마트 도어락 | **최종 보안문 앞** — 마지막 출구를 지키는 스마트 잠금 | 최종문 접근 |

### CH4 · 최종 탈출
- **메인 시스템 복구 & 탈출** (기존 미니 스마트홈 통합) — 조명·온도·도어·경보를 모두 복구 → **메인 보안문 해제** → EDDIE 탈출.
- **엔딩 연출**: 문 개방 + 빛 쏟아짐 + EDDIE 세리머니 + 수료 + 미스터리 떡밥.

## 5. 연출·분위기 가이드 (에셋용)
- 배경: 다크 네이비를 거의 블랙까지 + **비상등 앰버/레드 포인트 글로우**, 깜빡임, 긴 그림자, 먼지 입자, 비네팅.
- 조명: 미해결 방=어둡고 EDDIE 눈빛만. **클리어 시 그 구역만 환하게** 전환(전력 복구 연출).
- 사운드(참고): 낮은 험, 멀리서 삐걱, 클리어 시 안도의 차임. (실제 구현은 옵션)
- HUB(연구소 복도): 풀린 방 수에 따라 복도가 점점 밝아짐 = 탈출 진척 시각화.
