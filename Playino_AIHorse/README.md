# 🐎 AI 경주마 키우기 — EDDIE 코치

경주마를 키우는 게임으로 **AI 학습의 핵심 루프**를 체득시키는 교육용 웹게임 (프로토타입).

> 한 줄 컨셉: **데이터를 읽고 → 승률을 예측하고 → 어디에 투자할지 최적화하라.**

## 실행
빌드 불필요. `index.html`을 브라우저로 더블클릭(또는 드래그) 하면 바로 실행됩니다.

## 게임 루프와 박혀 있는 AI 개념
| 게임 | AI 개념 |
|---|---|
| 말의 능력치(속도·스태미나·가속·근성) | **특징(feature) 벡터** |
| 경기 유형별 중요도(가중치) | **모델 weight** — 점수 = Σ(가중치 × 능력치) (선형 모델) |
| EDDIE 승률 % 예측 | **모델 추론(prediction)** — softmax 기반 |
| 노란 **민감도 막대** (능력치 +3 시 승률 변화) | **경사(gradient)** — ∂승률/∂능력치. 효율 큰 곳에 투자 = 경사하강법 직관 |
| 한정 포인트로 능력치 투자 | **최적화(optimization)** — 손으로 하는 gradient ascent |
| 날씨/컨디션 랜덤 | **노이즈 / 불확실성** — 예측 ≠ 확정 |
| 경주 결과로 코치가 복기 | **피드백 루프** |
| 마지막 챔피언십(혼합 경기) | **일반화 vs 과적합** — 한 유형만 특화하면 불리 |

## 핵심 교육 포인트 (이 게임의 차별점)
플레이어가 **"승률을 가장 많이 올리는 능력치"**(=민감도 막대가 가장 긴 곳)에 투자하는 행동이
곧 **경사하강법(어디로 가야 손실이 줄어드나)** 의 직관입니다. 이미 높은 능력치는 효율이
떨어진다는 것(수확 체감)도 자연스럽게 체득됩니다.

## 현재 범위 (MVP)
- ✅ 핵심 루프: 예측 + 최적화(gradient) 중심 (요청한 "핵심 깊게")
- ✅ 6라운드 시즌 + 챔피언십, 경주 애니메이션, 코치 멘트
- ⬜ 다음: 능력치별 훈련 트레이드오프(부상=노이즈), 시즌 간 강화학습(AI 코치 vs 나) 모드
- ⬜ 다음: Phaser 포팅 + 그래픽/사운드, EDDIE 세계관 연동

## 파라미터 튜닝 위치 (`index.html` 상단)
- `RACE_TYPES` : 경기 유형별 가중치
- `TEMP` : 실력차가 승률에 반영되는 강도(softmax 온도)
- `TRAIN_STEP` / 라운드별 포인트 : 난이도
